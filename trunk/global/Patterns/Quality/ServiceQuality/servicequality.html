An ontology design pattern for (web) service quality characterization in the Semantic Sensor Web.

An ontology design pattern for (web) service quality characterization in the Semantic Sensor Web. (Ontology) VC Definition

The concepts suggested in this module are not aligned to a foundational ontology because the goal is to provide a simple and readily re-usable set of concepts for semantic annotation. The module aligned to the foundational ontology Dolce Ultra Light (DUL) can be accessed at http://ifgi.uni-muenster.de/~degbelo/Ressources/SSN2012Degbelo. The module is similar to the 'Ontology design pattern for spatial data quality characterization in the Semantic Sensor Web'.

Service Quality Component (OWL Class) Definition In use

A service quality component is any property of the service that the consumer wants to approximate. Examples of service quality components are the reliability or the robustness of the web service.

Definition

The URI of this class is http://www.semanticweb.org/ontologies/dataquality.owl#ServiceQualityComponent

Service Quality Criterion (OWL Class) Definition In use

A service quality criterion is a criterion defined by the service consumer as meaningful to get information about the quality of the web service. It is a measure that helps to approximate the value of the service quality component.

Definition

The URI of this class is http://www.semanticweb.org/ontologies/dataquality.owl#ServiceQualityCriterion
